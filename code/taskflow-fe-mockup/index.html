TaskFlow — UI Mockup

Click vào từng màn hình bên dưới để xem thiết kế trước khi code React.

01

Login / Register

Split-screen, illustration trái + form phải. Đăng nhập, đăng ký, quên mật khẩu.
02

Dashboard / Projects

Topbar + sidebar + grid cards project. Empty state, search, filter.
03

Kanban Board (chính)

Drag-drop task giữa các list, task card đẹp với label/assignee/due/checklist progress.
04

Task Detail Panel

Side panel slide-in: description, properties, checklist, comments, attachments, activity.
05

Notifications

Dropdown từ topbar + page riêng. Realtime badge.
06

Members / Settings

Quản lý thành viên, role dropdown, invite modal.

Nguyên tắc thiết kế

  • Minimal, breathing room — neutral gray + 1 accent indigo, không gradient lòe loẹt
  • Inter font — clean, modern, đọc dễ
  • Card subtle shadow — nhấn nhá hierarchy
  • Smooth transition — hover lift 1-2px, drag-drop animation 150ms
  • Skeleton loading thay vì spinner
  • Empty state có illustration + CTA
  • Optimistic UI — kéo task xong update ngay, fail thì rollback
  • Dark mode sẵn sàng (toggle ở topbar)
